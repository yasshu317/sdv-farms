Product guide

How people use this website

Visitors, buyers, sellers, and admin / staff each follow a clear path. This page is the business-facing flow document — share the live URL /business/flows on your domain, or distribute this HTML from the deployed site root.

Visitors Buyers Sellers Admin & staff Routing

Visitors No account

Anyone can explore marketing content, listings, services, and contact without signing in.

  1. Home page → story, gallery, location, Book a visit / contact.
  2. /properties — filter verified agriculture listings.
  3. Property detail → enquire / book visit (as offered on the page).
  4. /buyer-request — tell the team what land you need (guest or signed-in).
  5. /services — Phase II services and enquiries for landowners.
  6. Sign up only when a personal dashboard or seller tools are needed.

Buyers Role: buyer

The buyer dashboard tracks property enquiries, plot interests, and land requests.

  1. Register at /auth/register (choose Buyer) or sign in at /auth/login.
  2. After email verification (if required) → first home is /dashboard.
  3. Tabs such as Overview, Enquiries, Plot interests, Land requests show lead status (pending, contacted, visited, booked, etc.).
  4. Keep using /properties; actions on listings connect to the dashboard where supported.

Fulfilment is done by admin/staff in the operations hub.

Sellers Role: seller

Sellers manage listings from the seller hub; SDV reviews and approves before go-live.

  1. List your land → /auth/register?flow=seller (seller-specific signup).
  2. Eligibility check blocks restricted land types (e.g. Poramboke, Forest, etc.).
  3. After verify + login → /seller.
  4. Create /seller/property/new; edit via /seller/property/[id]/edit.
  5. Active listing limits and review rules apply in-product.

Admin & staff admin · staff

Operations share /admin. Access is role-gated; sellers use /seller, not admin.

  1. Sign in as admin or staff → land on /admin.
  2. Work enquiries, buyer requests, appointments, seller properties (review/edit), plots/verification, and other ops tabs exposed in the live admin UI.
  3. Admin-only tools may include user management, feature flags, and full listing control — match what you see at /admin.

After login — default home

  • Buyer (default) → /dashboard
  • Seller → /seller
  • Admin / staff → /admin
