## 1. What this app is

Bilingual (English + Telugu) agricultural land marketplace for Telangana / AP / Karnataka.
Two-sided: **sellers** list verified land, **buyers** browse, wishlist, book site visits.

**Stack:** Next.js 16 (App Router) · Tailwind CSS (custom `paddy-*` / `turmeric-*` palette) · Supabase (Postgres + RLS + Auth) · Resend email · Vercel hosting · Jest + Playwright tests.

## 2. Information architecture (agreed, do not change without discussion)

| Surface | Primary route | Who lands here |
|---------|--------------|----------------|
| Marketing / discovery | `/` (anon) · `/?stay=1` when signed in | Visitors, shareable links |
| Buyer hub | `/dashboard` | Buyers; sellers may also visit |
| Seller workspace | `/seller` | Sellers |
| Ops / admin | `/admin` | Admin + staff |

**Rule when adding new features:**
- Buyer-side actions (transactions, services, shortlist) → **`/dashboard`**
- Seller listing / lead mechanics → **`/seller`**
- Brochure / discovery content → marketing routes (`/properties`, `/services`, `/buyer-request`, home sections)
- Admin tooling → **`/admin`**
