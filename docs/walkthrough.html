🏡 Buyer Journey — "I want to invest in agricultural land"
1
Lands on home page
Sees the hero, chatbot for quick questions, "How It Works" explained in 3 steps. Language toggle for Telugu.
/
2
Browses property listings
Filters by State, Soil Type, Area, and Price. Each card shows acreage, price/acre, and location.
/properties
3
Opens a property detail page
Photo gallery, full land details, document type, road access, SDV-verified badge.
/properties/{id}
4
Books a site visit
Picks a date and 1-hour time slot (minimum 2-hour notice). Receives email confirmation with property and slot details.
Appointment picker on detail page
5
Registers & tracks (optional)
Creates a Buyer account. Dashboard shows all enquiries, plot interests, and their current "journey" (Registered → Enquired → Site Visit → Booked).
/dashboard
6
Can't find what they want?
Posts a land request — state, district, budget, area preference. Admin matches them with a suitable listing.
/buyer-request
🌾 Seller (Farmer / Agent) Journey — "I have land to sell"
1
Registers as Seller — chooses Farmer or Agent
Chooses "Sell Land" → eligibility check (Poramboke, Forest & restricted land types are blocked with a clear message) → selects sub-type:
 • Farmer — owns the land directly
 • Agent — represents a landowner
After registration the success screen shows "🌾 Seller (Farmer)" or "🌾 Seller (Agent)" so the user always knows their role.
/auth/register
2
Clicks "+ Add Property"
3-step form: Step 1 — State, District, Mandal, Village. Step 2 — Land type, soil type, area, price/acre, road access. Step 3 — Upload Pahani / ROR-1B / Adangal (Submit button disabled until document is attached).
/seller/property/new
3
Listing is "Pending Review"
SDV team reviews within 24 hours. Seller sees status clearly in their dashboard.
/seller
4
Admin approves → listing goes live
Auto-assigned SDV-YYYY-NNN property ID. Seller receives email. Listing is now public on /properties.
Status: Approved
5
Tracks performance
Sees view count, buyer enquiries, upcoming site visit appointments. Can edit the listing or add another (up to 2 active listings).
/seller
⚙️ Admin Journey — "SDV team managing the platform"
1
Logs in → Admin Panel
Automatically routed to the admin dashboard. Role badge visible in top-right corner.
/admin
2
Reviews & approves property listings
Properties tab: view submitted listings with all documents. Click Approve → system assigns SDV ID and notifies seller. Click Reject → listing is removed.
Admin → Properties
3
Manages enquiries & leads
Updates each lead status: Pending → Contacted → Site Visited → Booked → Closed.
Admin → Enquiries
4
Manages site visit appointments
Sees all booked slots. Can confirm or cancel. Buyer and seller both receive email on update.
Admin → Appointments
5
Matches buyer land requests
Reviews open buyer requests (state, budget, area), matches them to a listing, marks as Matched or Closed.
Admin → Requests
